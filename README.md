# 🍽️ La Buena Mesa - Landing Page de Restaurante

¡Bienvenido al repositorio de **La Buena Mesa**! Este proyecto es una landing page moderna y elegante diseñada para el sector gastronómico, enfocada en la experiencia de usuario y el impacto visual.

---

## 🌟 Características del Proyecto
- **Estética Gourmet:** Uso de tipografías clásicas (`Playfair Display`) combinadas con modernas (`Poppins`) para transmitir elegancia.
- **Secciones Clave:** Hero impactante, Sobre Nosotros, Menú Interactivo, Galería de imágenes y Contacto.
- **Diseño Responsive:** Adaptabilidad total para que los clientes puedan ver el menú desde sus celulares.
- **Efectos Visuales:** - Fondos con efecto *Parallax* (fijo al hacer scroll).
  - Hover dinámico en los ítems del menú con cambios de color y movimiento.
  - Galería simétrica con efectos de zoom al pasar el mouse.

---

## 🛠️ Tecnologías Utilizadas
- **HTML5:** Estructura semántica para mejorar el posicionamiento.
- **CSS3 Avanzado:** - **CSS Grid:** Utilizado para la galería simétrica y la grilla del menú.
  - **Flexbox:** Para la alineación de elementos y el contenido del Hero.
  - **Gradientes y Texturas:** Combinación de colores profundos (`#050a15`) con texturas de papel natural para la sección "Sobre Nosotros".
- **Google Fonts:** Integración de fuentes personalizadas.

---

## 📸 Vista Previa
El sitio cuenta con una galería de 6 imágenes optimizadas y una sección de contacto directa mediante un botón vinculado a WhatsApp, facilitando las reservas inmediatas.
